From 8/25/2020

What is the output of the following program? (src/click1.cpp) 

#include <cstdio>
#include <iostream>
#include <string>
using namespace std;

int main()
{
  string s1, s2;
  const char *s;

  s2 = "Fred";
  s = s2.c_str();
  s1 = s;

  s1[1] = 'y';
  s2[1] = 'z';
 
  cout << s << " "
       << s1 << " "
       << s2 << endl;
  return 0;
}

Multiple Choice Answers:

A. Fyed Fyed Fzed
B. Fred Fzed Fzed
C. Fyed Fyed Fyed
D. Fzed Fzed Fzed
E. Fred Fyed Fzed
F. Fzed Fyed Fzed
G. Fred Fred Fred
H. Fred Fyed Fzed

Here's the answer (F): 

UNIX> make bin/click1
g++ -std=c++98 -Wall -Wextra -o bin/click1 src/click1.cpp
UNIX> bin/click1
Fzed Fyed Fzed
UNIX>
Why? You start with s2 being "Fred", and s pointing to the underlying string: 

s2: "Fred"
    ^
    |
s ---

When you assign s1 to s, it makes a copy -- C++ strings always make copies: 

s2: "Fred"
    ^
    |
s ---
s1: "Fred"

Now, each string changes its second character -- s1 changes it to 'y' and s2 changes it to 'z': 

s2: "Fzed"
    ^
    |
s ---
s1: "Fyed"

Thus, the output is "Fzed Fyed Fzed".
