Priority Queue Clicker Questions

Let us suppose that you implement a priority queue using a multiset. Answer these four questions:
  1. What is the running time of Push() when the priority queue has n elements?
  2. What is the running time of Pop() when the priority queue has n elements?
  3. What is the running time of Size() when the priority queue has n elements?
  4. What is the running time of creating a priority queue from a vector with n elements?
