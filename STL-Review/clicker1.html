From 8/27/2020

In the following procedure, assume that the size of p is m, and the size of r is o. What is the running time of this function: 

int find_em(const vector <int> &r,
            const map <int, int> &p,
            int n) 
{
  int found;
  int i, index;

  found = 0;
  for (i = 0; i < n; i++) {
    index = rand() % r.size();
    if (p.find(r[index]) != p.end()) found++;
  }
  return found;
}

Multiple Choice Answers 

A. O(m log(m))
B. O(m log(n))
C. O(m log(o))
D. O(n log(m))
E. O(n log(n))
F. O(n log(o))
G. O(o log(m))
H. O(o log(n))
I. O(o log(o))

The loop iterates n times, and since the map is size m, each find() operation is O(log(m)). So the answer is D: O(n log(m)).
