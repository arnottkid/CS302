General Graph Clicker Questions

An undirected graph is defined as follows:

V = { i | 0 <= i < 10 }
E = { (0,1), (0,3), (1,9), (3,5), (3,7), (4,6), (5,9), (6,8) }

    1. How many connected components are in the graph?

    Answer: The easiest way to answer this is to draw the graph: 

    0 ------- 1           2
|          \_______
|                  \
3 ------- 5 -------- 9
|
|
7       4---6---8
    The answer is three.

    2. Which nodes are in the one cycle in this graph?

    Answer: Look at the graph above: 0, 1, 3, 5, 9.

    3. Is this graph bipartite?

    No -- it has an odd cycle, and if you think about it, an odd cycle cannot be in a bitpartite graph.
